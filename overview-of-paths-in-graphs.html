Note that the definition of an Hamiltonian graph says nothing about whether its edges are distinct, however this is implied by the fact that the path visits all vertices exactly once. Also my definition of distinct edges does not include the first and last vertices for closed paths.

From the table a number of statements are obvious:
  • A cycle is a simple path
  • A traversable path is a trail
  • An Eulerian path is a traversable path, and also a trail
  • An Hamiltonian path is a cycle
  • An Hamiltonian path is a trail
  • An Hamiltonian path, an Eulerian path and a traversable path are all spanning paths
  • An Eulerian path may use a vertex more than once
  • An Hamiltonian path may not use all edges
